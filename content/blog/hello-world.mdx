# 欢迎来到我的博客

这是我的第一篇博客文章！这个博客使用 **Next.js** + **MDX** 构建，具有以下功能：

## 功能特性

### 1. Markdown 支持
- 支持标准 Markdown 语法
- 支持代码高亮
- 支持表格、列表等

### 2. 评论系统
使用 Giscus 评论系统，基于 GitHub Discussions：
- 免费、开源
- 支持 Markdown
- 支持表情反应

### 3. 搜索功能
使用 Fuse.js 实现模糊搜索：
- 支持中文搜索
- 模糊匹配
- 实时搜索建议

### 4. SEO 优化
- 自动生成 Sitemap
- 自动生成 RSS Feed
- 完善的 Meta 标签

## 开始使用

1. 在 `content/blog/` 目录下创建 `.mdx` 文件
2. 添加 front matter（标题、日期、标签等）
3. 使用 Markdown 编写内容
4. 保存后自动更新
