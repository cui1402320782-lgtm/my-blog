# Markdown 使用指南

这是一篇使用 **.md** 格式的文章，不是 .mdx 格式。

## 什么是 Markdown

Markdown 是一种轻量级标记语言，它允许你使用易读易写的纯文本格式编写文档。

## 基本语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

```markdown
**粗体**
*斜体*
~~删除线~~
```

### 列表

无序列表：
- 项目 1
- 项目 2
- 项目 3

有序列表：
1. 第一步
2. 第二步
3. 第三步

### 链接和图片

```markdown
[链接文字](https://example.com)
![图片描述](image.jpg)
```

### 代码

行内代码：`console.log('hello')`

代码块：

```javascript
function greet(name) {
  return `Hello, ${name}!`
}
```

### 引用

> 这是一段引用文字。
> 可以有多行。

### 表格

| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |

## 结语

Markdown 简单易学，是编写博客内容的理想选择！
